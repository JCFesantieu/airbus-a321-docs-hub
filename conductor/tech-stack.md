# Technology Stack

This document outlines the core technologies and frameworks used in the project.

## Overview

The project leverages a modern web development stack designed for high performance, scalability, and developer experience.

## Frontend

*   **Framework:** Next.js 16+ (React)
    *   **Description:** A React framework for building full-stack web applications, enabling server-side rendering (SSR), static site generation (SSG), and API routes.
*   **Language:** TypeScript 5+
    *   **Description:** A superset of JavaScript that adds static typing, improving code quality and maintainability.
*   **UI Components:** Shadcn UI (built on Radix UI)
    *   **Description:** A collection of accessible and customizable UI components that integrate seamlessly with Tailwind CSS.
*   **Styling:** Tailwind CSS 4+
    *   **Description:** A utility-first CSS framework for rapidly building custom designs.

## Libraries

*   **Search:** Fuse.js 7+
    *   **Description:** A powerful, lightweight fuzzy-search library for client-side searching.
*   **Animation:** Framer Motion 12+
    *   **Description:** A production-ready motion library for React.
